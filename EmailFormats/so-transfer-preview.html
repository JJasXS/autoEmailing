Run: dotnet run (default test page on port 2086, if terminal is interactive).
Optional explicit preview mode: dotnet run -- --preview-so; optional alternate port: PREVIEW_SO_PORT=2086 (or another free port).
If you open the preview using the EC2 public DNS or IP and get 400 Invalid Hostname, set PREVIEW_SO_ANY_HOST=1 in .env and on Windows run once (elevated): netsh http add urlacl url=http://+:2086/ user=Everyone (adjust port if needed).
Layout matches the ERP reference: a bold group row per SO (SO-… + company), then columns Seq., Code, Description, U/Price, Delivy date, Orig Qty, O/Stding, and transfer Date / Doc No / Tfer Qty (extra transfers as sub-rows with the first seven columns blank). Outstanding = SL_SODTL.SQTY - SUM(ABS(ST_XTRANS.SQTY)) <> 0 for FROMDOCTYPE='SO'. Line Seq. uses ROW_NUMBER() OVER (PARTITION BY SO.DOCKEY ORDER BY DTL.DTLKEY); unit price from SL_SODTL.UNITPRICE.
